# Online-Shoppers-Purchase-Prediction

## Overview

This project predicts whether a website visitor is likely to make a purchase based on browsing behavior and session information.

## Dataset

The dataset contains visitor information such as page visits, time spent on pages, traffic source, operating system, browser details, and visitor type.

## Workflow

* Data Cleaning
* Exploratory Data Analysis
* Feature Encoding
* Feature Scaling
* Model Building
* Model Evaluation

## Models Used

* Logistic Regression
* Random Forest
* XGBoost

## Objective

To identify factors that influence purchasing behavior and build a classification model that predicts customer conversion.

## Technologies

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn

## Key Learning

This project improved understanding of customer behavior analytics, classification algorithms, and business-focused machine learning applications.
